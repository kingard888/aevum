# Contributors

AevumDB is developed and maintained by its community. This file acknowledges everyone who has contributed to the project.

## Core Team

- **Developers**: The AevumDB development team
- **Maintainers**: Active project maintainers

## Contributors

We welcome all contributions! Contributors include:

- **Code**: Bug fixes, features, performance improvements, and refactoring
- **Documentation**: Guides, API documentation, examples, and translations
- **Testing**: Bug reports, test cases, and quality assurance
- **Community**: Discussion leadership, mentoring, and advocacy

### How to Get Listed

When you submit a pull request that gets merged, you'll automatically be credited as a contributor on GitHub.

To manually update this file with specific names or roles, please include it in your contribution or create an issue.

## Acknowledgments

AevumDB builds on the work of several open-source projects:

- **WiredTiger**: High-performance storage engine (http://source.wiredtiger.com/)
- **MongoDB C Driver**: BSON serialization library (https://github.com/mongodb/mongo-c-driver)
- **simdjson**: Fast JSON parsing (https://simdjson.org/)

Special thanks to the MongoDB project for inspiring the query interface design.
