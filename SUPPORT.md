### Ask the Community

1. **GitHub Discussions**: https://github.com/aevumdb/aevum/discussions
   - Best for questions and general discussion
   - Community members help answer questions
   
2. **GitHub Issues**: https://github.com/aevumdb/aevum/issues
   - Use only for bugs or feature requests
   - Not recommended for general questions

3. **Email**: For important concerns, email the team (contact through GitHub)

## Reporting Issues

Found a bug? Please report it:

1. Check if it's already reported: https://github.com/aevumdb/aevum/issues
2. Provide details:
   - **Error message**: Exact error output
   - **Steps to reproduce**: Commands that trigger the issue
   - **Version**: Output of `./build/bin/aevumdb --version`
   - **Environment**: OS, CPU, RAM info
3. Include logs if available

See [docs/TROUBLESHOOTING.md#reporting-a-bug](docs/TROUBLESHOOTING.md#reporting-a-bug) for detailed reporting guidelines.
